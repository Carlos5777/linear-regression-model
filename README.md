# linear-regression-model
# Linear Regression Model

## Overview
This project is a simple implementation of a Linear Regression model using Python  
The goal is to understand how basic regression works and how it can be used to make predictions from numerical data

## What it does
The model takes a set of input features and learns how they relate to a target variable  
After training, it can predict new values based on unseen data

## Tools and libraries
- Python
- pandas
- scikit-learn

## Files in this repository
- `model.ipynb` → main notebook with the full process (data loading, training, evaluation)
- `model.py` → script version of the model
- `requirements.txt` → dependencies needed to run the project
